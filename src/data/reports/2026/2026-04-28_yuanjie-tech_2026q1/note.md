# 源杰科技 2026 年第一季度报告解读

## 这份报告能支持什么

源杰科技 2026 年第一季度报告披露，公司 2026Q1 营业收入同比增长 320.94%，归属于上市公司股东的净利润同比增长 1153.07%。

报告在“主要会计数据、财务指标发生变动的情况、原因”中说明，增长主要受益于客户需求增长，数据中心领域 CW 光源产品销售额实现增长，公司整体产品结构进一步优化；同时数据中心产品毛利率较高，带动净利润水平同比增加。

因此，这份报告可以支持当前图谱中的：

- `cw_laser` 节点：CW 激光器 / CW 光源
- `cw_laser -> optical_engine` 边：激光光源供应

## 映射口径

报告没有披露“数据中心 CW 光源”的单独收入金额，也没有披露每个客户或每条链路的交易额。因此图谱中使用公司整体营业收入同比增长作为 `cw_laser -> optical_engine` 的增长代理。

这个代理比直接披露的边级交易金额弱，但报告原文明确把业绩增长原因指向“数据中心领域 CW 光源产品销售额实现增长”，因此比单纯用公司整体收入硬套要更有支撑。

## 关键数据

- 2026Q1 营业收入：355,278,486.49 元，同比 +320.94%。
- 2026Q1 利润总额：211,353,903.33 元，同比 +1151.30%。
- 2026Q1 归母净利润：179,441,968.61 元，同比 +1153.07%。
- 2026Q1 扣非归母净利润：177,939,676.39 元，同比 +1173.64%。
- 2026Q1 经营活动现金流净额：39,650,807.07 元。
- 2026Q1 研发投入：20,650,110.32 元，同比 +77.94%。

## 局限性

这份报告不能直接支持 CPO 模组、可插拔光模块、交换机或 AI 服务器整机的边级增长。

报告也没有披露“数据中心产品利润”“数据中心 CW 光源毛利”或对应同比增速。它只定性说明数据中心领域 CW 光源产品销售额增长、数据中心产品毛利率较高，从而带动公司净利润水平同比增加。因此不能把归母净利润同比 +1153.07% 直接当作数据中心 CW 光源利润同比增长。

当前只把它作为上游 CW 光源对光引擎链路的增长信号，不向下游链路继续传导，避免把上游芯片收入增长重复计入光模块或系统收入。

原始 PDF、东方财富页面和公告内容接口分页缓存均保存在 `raw/`，该目录被 Git 忽略；结构化结果保存在同目录的 `extracted.json`。
